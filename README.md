# 🚀 XWorm RAT - Advanced Remote Access Tool

![Version](https://img.shields.io/badge/version-1.0-blue?style=for-the-badge)

📌 Описание
XWorm RAT — это мощный инструмент удалённого администрирования, разработанный на C# (.NET Framework 4.6+). Он предоставляет полный контроль над удалённой системой через зашифрованное C2-соединение. Идеально подходит для тестирования систем безопасности и образовательных целей.
🚀 Возможности

🖥️ Управление системой
-  Полный доступ к файловой системе (загрузка, скачивание, удаление)
-  Удалённый рабочий стол (скриншоты, веб-камера)
-  Управление процессами и службами
-  Выполнение команд CMD / PowerShell
🔐 Сбор данных
  Кейлоггер (с поддержкой раскладок)
  Кража паролей из браузеров (Chrome, Edge, Firefox, Opera)
  Сбор cookie-файлов и сессий
  Извлечение криптовалютных кошельков
  Сбор информации о системе (ОС, CPU, GPU, RAM, антивирусы)
🧩 Расширяемость
  Поддержка плагинов (загрузка и выполнение внешних модулей)
  Обновление бинарника через C2-команду update
  Подгрузка .NET сборок прямо в память
🌐 Сеть и персистенция
  Автозапуск через реестр / планировщик задач
  Распространение через USB-накопители
  Поддержка обратного прокси (FRP / ngrok)
  Шифрованный трафик (AES-256)
🛡️ Скрытность
  Анти-отладка, анти-сборка (обфускация)
  Поддержка VPS и динамических DNS
  Обход Windows Defender через AMSI-патчинг

🧰 Используемые технологии
| Компонент | Назначение |
|------------------------|
|.NET Framework 4.6+|Платформа выполнения|
|------------------------|
|AES-256 + RSA|Шифрование трафика и конфигурации|
|------------------------|
|Newtonsoft.Json|Сериализация данных|
|------------------------|
|.NET Reactor / ConfuserEx	|Обфускация кода|
|------------------------|
|FRP / ngrok|Проксирование трафика|
